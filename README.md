# Exo MR Dupont — Atelier Dépannage Windows 10

## 📖 Description

Cette machine virtuelle a été conçue comme un atelier pratique de dépannage Windows 10.

Le but est de se mettre dans la peau d’un technicien support/administrateur système et de diagnostiquer plusieurs problèmes présents sur le poste de travail de “M. Dupont”.

L’environnement contient différentes pannes réalistes inspirées de situations rencontrées en support informatique.

---

# 🧩 Scénario

> Bonjour,
>
> Depuis quelques jours, mon ordinateur fonctionne très mal.
>
> Il est devenu extrêmement lent et des fenêtres étranges apparaissent régulièrement à l’écran.
>
> En plus, mon imprimante ne fonctionne plus du tout, et certaines photos ont disparu de mon dossier “Images”.
>
> J’ai aussi remarqué qu’un disque semblait avoir un problème dans Windows.
>
> Enfin, depuis ce matin, Windows ne démarre plus correctement…
>
> Pouvez-vous diagnostiquer et réparer tous les problèmes ?
>
> Merci beaucoup !
>
> — M. Dupont

---

# 🎯 Objectifs

Le participant devra :

- Réparer le démarrage de Windows
- Diagnostiquer les lenteurs système
- Identifier et supprimer les processus parasites
- Réparer le système d’impression
- Retrouver les fichiers disparus
- Vérifier l’état des disques
- Restaurer un fonctionnement normal du poste

---

# 🛠️ Compétences travaillées

- Réparation du boot Windows
- Utilisation des outils de récupération Windows
- Analyse des performances système
- Gestion des tâches planifiées
- Gestion des services Windows
- Gestion des fichiers cachés
- Gestion des disques
- Diagnostic et méthodologie de troubleshooting

---

# 💻 Outils utiles

Le challenge peut être résolu avec les outils suivants :

- Gestionnaire des tâches
- Services Windows (`services.msc`)
- Gestion des disques (`diskmgmt.msc`)
- Planificateur de tâches (`taskschd.msc`)
- Invite de commandes
- PowerShell
- Outils Sysinternals
- Outils de récupération Windows
